AWK和SED文本处理常用命令汇总

2019年1月1日
| linux
| 阅读

使用awk过滤输出：

docker ps|awk 'NR!=1 {print $1}'
# 作用：过滤docker ps输出的结果，只显示第一列Container ID的值，且不显示第一行”Container ID“
# NR!=1表示过滤掉第一行，NR==1表示只匹配第一行。END表示最后一行
# awk默认以空格为分隔符对每一行进行切片，$0表示所有内容，$1表示（以空格分割的）第一列。$NF表示最后一列
# eg:打印file1中最后一行最后一列
awk 'END {print $NF}' file1.txt

使用sed处理文件内容：

-i 表示直接修改文件中的内容，如果没有-i则只将处理后的内容输出到console，但原文件中内容未改变。

# 删除文件中空白行
sed -i '/^$/d' file1.txt
# 删除文件中最后一行的内容
sed -i '$d' file1.txt
# 删除文件中第一行到第三行（注意，共删除1，2，3三行的内容）
sed -i '1,3d' file1.txt

# 向文件中第3行后面追加一行内容hello world
#（append模式，原文件第4行变为第5行，依次类推……）
sed -i '3a hello world' file1.txt
# 向文件第1行插入内容。insert模式。原文件第1行变为第2行
sed -i '1i hello world' file1.txt

# 将文件中第2到第5行的内容全部替换为hello world
sed -i '2,5c hello world' file1.txt

# 将文件中所有hello替换为goodbye，g代表替换所有匹配到的值，没有则只匹配并替换第1个
sed -i 's/hello/goodbye/g' file1.txt

# 输出文件第2到第5行
sed -n '2,5p' file1.txt
