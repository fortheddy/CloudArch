安装配置Prometheus + Grafana运维监控平台

一、安装配置Prometheus

1、下载Prometheus和node_exporter安装包并解压

wget https://github.com/prometheus/prometheus/releases/download/v2.8.1/prometheus-2.8.1.linux-amd64.tar.gz
tar -xvzf prometheus-2.8.1.linux-amd64.tar.gz
wget https://github.com/prometheus/node_exporter/releases/download/v0.17.0/node_exporter-0.17.0.linux-amd64.tar.gz
tar -zxvf node_exporter-0.17.0.linux-amd64.tar.gz

2、修改prometheus.yml文件

scrape_configs:
  # The job name is added as a label `job=<job_name>` to any timeseries scraped from this config.
  - job_name: 'prometheus'

    # metrics_path defaults to '/metrics'
    # scheme defaults to 'http'.

    static_configs:
    - targets: ['192.168.1.135:9090']

  - job_name: node_exporter
    static_configs:
    - targets: ['192.168.1.135:9100']

prometheus默认运行在9090端口，node_exporter默认运行在9100端口

targets中为被监控的机器节点，可以并列写入多个节点，用逗号分隔。

3、启动prometheus和node_exporter

# 方法a、直接后台启动（停止时需要通过ps命令找到进程id后手动kill）

nohup ./prometheus --config.file=prometheus.yml &
nohup ./node_exporter &

# 方法b、注册系统服务

vim /etc/systemd/system/prometheus.service

[Unit]
Description=Prometheus Server
Documentation=https://prometheus.io/docs/introduction/overview/
After=network-online.target

[Service]
Restart=on-failure
ExecStart=/data/prometheus-2.8.0.linux-amd64/prometheus --config.file=/data/prometheus-2.8.0.linux-amd64/prometheus.yml

[Install]
WantedBy=multi-user.target

vim /etc/systemd/system/node_exporter.service

[Unit]
Description=node_exporter
After=network.target

[Service]
Restart=on-failure
ExecStart=/data/node_exporter-0.17.0.linux-amd64/node_exporter

[Install]
WantedBy=multi-user.target

启动服务

systemctl daemon-reload
systemctl start prometheus
systemctl start node_exporter

4、浏览器登录http://192.168.1.135:9090/graph

二、安装配置Grafana

1、下载并安装Grafana

wget https://dl.grafana.com/oss/release/grafana-6.1.0-1.x86_64.rpm
sudo yum localinstall grafana-6.1.0-1.x86_64.rpm

2、启动grafana

systemctl start grafana-server

3、浏览器登录http://192.168.1.135:3000/login ，初始登录时会有一个默认用户名密码admin/admin
